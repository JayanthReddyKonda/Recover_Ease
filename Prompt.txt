# WHAT YOU ARE BUILDING

A post-discharge patient recovery monitoring system with:
- Patients log daily symptoms (voice/text/manual)
- Doctors monitor patients via intelligent triage dashboard
- Hybrid AI escalation engine (rules + Groq LLM) alerts doctors only when truly needed
- Real-time Socket.io alerts
- Doctor sends connection request to patient by Patient ID
- Patient accepts/rejects — only then data is shared
- Production-grade UI with Framer Motion animations

```typescript
import { Server, Socket } from 'socket.io';

export function initSocket(io: Server): void {
  io.on('connection', (socket: Socket) => {
    console.log(`Socket connected: ${socket.id}`);

    socket.on('join_doctor_room', (doctorId: string) => {
      socket.join(`doctor:${doctorId}`);
      socket.join('doctor_alerts');
      socket.data.doctorId = doctorId;
      console.log(`Doctor ${doctorId} joined rooms`);
    });

    socket.on('join_patient_room', (patientId: string) => {
      socket.join(`patient:${patientId}`);
      socket.data.patientId = patientId;
      console.log(`Patient ${patientId} joined room`);
    });

    socket.on('ping', () => socket.emit('pong'));

    socket.on('disconnect', () => {
      console.log(`Socket disconnected: ${socket.id}`);
    });
  });
}
```

## FILE: backend/src/services/escalation.service.ts

Import: prisma, groqService, uuid, Server from socket.io

FUNCTION — checkRules(logs: any[]): RuleCheckResult

Input: logs sorted by logDate ASC with medicationTaken parsed from JSON.
Check ALL rules, collect every triggered one:

RULE IMMEDIATE_SPIKE: any log with pain >= 9
RULE PAIN_SUSTAINED: pain > 8 for 3+ consecutive days (find max consecutive run)
RULE SLEEP_DEPRIVATION: sleepHours < 4 for 5+ consecutive days
RULE MULTI_SYMPTOM: fatigue > 8 AND swelling > 8 on same day
RULE MEDICATION_NON_COMPLIANCE: ALL meds false for 3+ consecutive days
RULE RAPID_DETERIORATION: any metric increases 4+ points in 2 consecutive days
RULE RECOVERY_PLATEAU: 14+ days logged, last 7 days avg pain within 1 point of first 7 days avg, patient past day 10

Return { triggered: rules.length > 0, rules, reason: rules.join('; ') }

FUNCTION — runEscalationCheck(patientId: string, io: Server): Promise<EscalationResult>

1. Fetch last 14 logs from Prisma ordered ASC, parse medicationTaken JSON on each
2. If < 2 logs: return { escalated: false, severity: 'NORMAL' }
3. Run checkRules
4. If not triggered: return { escalated: false, severity: 'NORMAL' }
5. Call groqService.runAIEscalationCheck (try/catch fallback to MONITOR)
6. Severity logic:
   CRITICAL = rules triggered AND ai verdict ALERT
   MONITOR = rules triggered AND ai verdict MONITOR
   NORMAL = ai verdict NORMAL (overrides rules)
7. Prisma create escalation record
8. If CRITICAL or MONITOR: emit to 'doctor_alerts' room:
   { patientId, severity, reason: aiResult.reason, isSOS: false, timestamp }
9. Call checkAndAwardMilestones(patientId, logs) — don't await
10. Return EscalationResult

FUNCTION — triggerSOS(patientId: string, io: Server): Promise<EscalationResult>

Create critical escalation with isSOS=true.
Emit immediately to 'doctor_alerts' with isSOS: true.
Return result.

FUNCTION — checkAndAwardMilestones(patientId: string, logs: any[]): Promise<void>

Check and upsert these milestones using Prisma:
- first_log: logs.length >= 1 → "First symptom logged! Your journey begins." 🌱
- week_streak: last 7 log dates all consecutive → "7-day logging streak! Perfect consistency." 🔥
- pain_improvement_5: 5 consecutive logs with decreasing pain → "5 days of reducing pain!" 📉
- sleep_champion: 5 consecutive logs sleepHours >= 7 → "5 nights of great sleep!" 😴
- pain_halved: last pain <= first pain / 2 → "Pain reduced by half since day 1!" 🎯
- med_compliance_week: last 7 logs all meds true → "Perfect medication week!" 💊

## FILE: backend/src/services/recovery.service.ts

FUNCTION — detectRecoveryStage(dischargeDate: Date, logs: any[]): RecoveryStageInfo

daysSince = floor((now - dischargeDate) / 86400000)
avgPainRecent = avg pain of last 5 logs

If daysSince <= 7 OR avgPainRecent > 7: ACUTE
  label: "Week 1 — Acute Recovery"
  color: "red"
  guidance.title: "Focus on Rest & Wound Care"
  guidance.instructions: ["Rest as much as possible", "Take all medications exactly on schedule", "Keep surgical wound clean and dry", "Apply ice 20 minutes every 2 hours for swelling", "Eat protein-rich foods to support tissue healing"]
  guidance.restrictions: ["No strenuous physical activity", "Do not get wound area wet", "Avoid alcohol completely", "No driving until cleared by doctor"]
  guidance.watchFor: ["Fever above 38°C / 100.4°F", "Increasing redness or warmth around wound", "Pus or unusual discharge", "Sudden severe pain increase"]

If daysSince <= 14 OR avgPainRecent > 4: SUBACUTE
  label: "Week 2 — Subacute Phase"
  color: "yellow"
  guidance.title: "Gentle Movement & Nutrition"
  guidance.instructions: ["Begin gentle walking 5-10 minutes twice daily", "Continue all prescribed medications", "Focus on high-protein, anti-inflammatory diet", "Aim for 7-9 hours of sleep each night", "Practice deep breathing exercises to prevent complications"]
  guidance.restrictions: ["No lifting over 2kg", "Avoid high-impact activities", "Do not skip physiotherapy appointments"]
  guidance.watchFor: ["Pain worsening after gentle activity", "New or increasing swelling", "Difficulty bearing weight that is not improving"]

Else: REHABILITATION
  label: "Week 3+ — Rehabilitation Phase"
  color: "green"
  guidance.title: "Progressive Strength & Activity"
  guidance.instructions: ["Increase walking distance and duration gradually each day", "Follow your physiotherapy exercise program consistently", "Return to light daily activities as tolerated", "Stay hydrated — minimum 2 litres of water daily", "Track your progress and celebrate improvements"]
  guidance.restrictions: ["Avoid high-impact sports for minimum 6 weeks", "Stop immediately if you feel sharp or sudden pain"]
  guidance.watchFor: ["Plateau in improvement after 2 weeks of rehab", "Swelling recurring with increased activity", "New symptoms not present in early recovery"]

FUNCTION — calculateRiskScore(logs: any[], escalations: any[]): { score: number, level: string }

Weighted factors:
- avgPain last 7 days / 10 * 35
- (10 - avgSleep last 7 days) / 10 * 20
- avgFatigue last 7 days / 10 * 20
- activeEscalations count * 5 (max 15)
- (missedMedDays / 7) * 10

score = Math.min(Math.round(sum), 100)
level: score < 40 = 'low', < 70 = 'medium', else 'high'
Return { score, level }
